Signal Reference Guide

Signal Methods

set(value: T): void;

Directly set the signal to a new value, and notify any dependents.

update(updateFn: (value: T) => T): void;

Update the value of the signal based on its current value, and notify any dependents.

mutate(mutatorFn: (value: T) => void): void;

Update the current value by mutating it in-place, and notify any dependents.

asReadonly(): Signal <T>;

Returns a readonly version of this signal. Readonly signals can be accessed to read their value but can't be changed using set, update or mutate methods. The readonly signals do _not_ have any built-in mechanism that would prevent deep-mutation of their value.
